# CCNA → CCIE Data Center

Этот репозиторий предназначен для изучения сетевых технологий от базового до экспертного уровня:

- **CCNA** – основы сетей, IP-адресация, VLAN, маршрутизация, базовая безопасность
- **CCNP** – продвинутая маршрутизация и коммутация, QoS, VPN, troubleshooting
- **CCIE Data Center** – Nexus/ACI, VXLAN, EVPN, Multi-VRF, Storage Networking, автоматизация

Каждая тема содержит:

- `theory/` – теория на русском языке
- `labs/` – лабораторные работы с конфигурациями и скриптами на английском языке

## Как использовать репозиторий

1. Изучаем теорию в папке `theory/`
2. Выполняем лабораторные работы в папке `labs/`
3. Вносите свои изменения через git и создавайте новые коммиты
